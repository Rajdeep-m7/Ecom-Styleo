# 🛍️ Styleo – E-Commerce Website

🌐 **Live Website:** https://styleo.onrender.com  

Styleo is a full-stack e-commerce web application built using the **MERN stack**, styled with **Tailwind CSS**, and powered by **Redux Toolkit** for state management.

It provides a complete shopping experience including authentication, cart management, and secure payments using **PayPal Sandbox**.

---

## 🚀 Tech Stack

### 🎨 Frontend
- React.js
- Redux Toolkit
- Tailwind CSS
- Axios
- PayPal React SDK

### 🖥️ Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
- JWT Authentication
- bcrypt.js

---

## ✨ Features

- User Registration & Login
- JWT Authentication
- Browse Products
- Product Details Page
- Add to Cart
- Update / Remove Cart Items
- Checkout System
- PayPal Sandbox Payment Integration
- Order History
- Admin Product Management
- Fully Responsive Design

---

## 💳 Payment Integration

This project uses **PayPal Sandbox** for testing payments.

- Secure order creation
- Client-side PayPal button integration
- Server-side payment verification
- Order status update after successful transaction

> ⚠️ Sandbox mode is used for development/testing. No real money is processed.

---

## 🔐 Security

- Password hashing with bcrypt
- JWT-based authentication
- Protected routes middleware
- Secure PayPal order validation
- Environment variable configuration

---

## 📦 Deployment

- Hosted on Render
- Database on MongoDB Atlas
- Payment Integration via PayPal Sandbox
